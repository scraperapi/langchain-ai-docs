---
title: "Chat models"
---

[Chat models](/oss/langchain/models) are language models that use a sequence of [messages](/oss/langchain/messages) as inputs and return messages as outputs (as opposed to using plain text). These are generally newer models.

<Info>
If you'd like to contribute an integration, see [Contributing integrations](/oss/contributing#add-a-new-integration).
</Info>

## Install and use

<AccordionGroup>
<Accordion title="OpenAI">

<CodeGroup>
```bash pip
pip install -qU "langchain[openai]"
```

```bash uv
uv add langchain[openai]
```
</CodeGroup>
```python
import getpass
import os

if not os.environ.get("OPENAI_API_KEY"):
  os.environ["OPENAI_API_KEY"] = getpass.getpass("Enter API key for OpenAI: ")

from langchain.chat_models import init_chat_model

model = init_chat_model("gpt-4o-mini", model_provider="openai")
```
</Accordion>
